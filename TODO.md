Todo App:
1. Login page.
    - this page will be used to login.
    - email & password will be used
    - Will also have option to use google sign in
2. List tasks
    - In this page we will be able to add task, delete task, postpone task
    - Sort tasks and display according to date,time

3. Task details
    - This page will show expanded detail of individual tasks
    - This page will also be used to edit information about task details
    - Show Date
    - Show Time
    - Task type - todo / event
    - Set finish Time
    - Notification/Timer
4. Calender view
    - will show calendar
    - will show tasks in given dates
5. Events page
    - This page will show all the tasks that are categorised as event
    - event sorted by dates
6. Task history
    - Will show completed tasks
    - option read the tasks back to the queue

7. Settings page:
    - Profile
    - Notifications
    - Theme
    - Logout
